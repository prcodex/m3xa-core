# Source tiering

> Not all sources are equal at every task. A wire is best for *breaking events*; an independent expert is best for *interpretation*; an institutional source (Bank1, Bank2) is best for *consensus framing*. The "both sides" rule says: when the topic is geopolitical, at least one source from each side of the narrative must be in retrieval.

This is a stub. Full essay to follow. Topics it will cover:

1. The three-tier hierarchy: Wire / Expert / Institutional — when each leads, when each lags.
2. The both-sides rule as a *retrieval* constraint (not a synthesis afterthought).
3. How tiers map onto retrieval scoring weights (`config/retrieval_scoring.yaml`).
4. Why state-aligned media is in the tier mix at all (it isn't "objective," but it's necessary for adversarial framing).
5. Anti-pattern: weighting all sources equally produces a confident-but-flat answer.
